# Typewriter Notes

A desktop app prototype for an old-school, distraction-free writing tool built with Electron.

## Run Locally

```sh
npm install
npm run start
```

## Prototype Coverage

- Desktop app shell with note library, note list, paper editor, toolbar, inspector, and status bar
- Sample notes for Morning Pages, Chapter Draft, Meeting Notes, Essay Outline, and Ideas to Revisit
- New note creation, note selection, autosave, live word and character counts
- Focus mode, typewriter sound toggle, theme selector, export dialog, and preferences window
- Local persistence through `localStorage`
- Shortcuts for `Cmd/Ctrl+N`, `Cmd/Ctrl+F`, `Cmd/Ctrl+S`, `Cmd/Ctrl+E`, and `Esc`
